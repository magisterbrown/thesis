\chapter{Introduction}
\label{ch:introduction}

\section{Motivation}

Nonuniform Fast Fourier Transforms (NUFFTs) are central to a wide range of scientific computing applications, including magnetic resonance imaging (MRI) reconstruction, radio astronomy, and molecular dynamics. FINUFFT is a state-of-the-art library for computing type 1, 2, and 3 NUFFTs in one, two, and three dimensions. Despite its strong baseline performance, FINUFFT exposes a number of configuration parameters whose optimal values depend heavily on the input size, dimension, and required accuracy.

Current practice typically relies on default parameter values or manual tuning by expert users, both of which leave significant performance on the table. Furthermore, several parameters that influence performance are not exposed to the user at all, instead being fixed to compile-time or runtime defaults determined without regard to the specific input. This thesis investigates the parameter space of FINUFFT systematically, covering both its tunable and its hardcoded parameters, to develop heuristics that minimize planning time while approaching optimal execution time.

\section{Research Questions}

This thesis addresses the following research questions:

\begin{enumerate}
    \item Which FINUFFT configuration parameters have the most significant impact on execution time, and under what input conditions?
    \item How should the spreading parameter $\sigma$ be selected to balance accuracy and performance for a given input?
    \item Can a lightweight heuristic be developed that selects near-optimal values for all tunable parameters without incurring significant planning overhead?
\end{enumerate}
