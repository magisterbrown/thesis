\section{Overview}

Before attempting to optimize FINUFFT's parameters, it is necessary to understand which parameters actually matter and to what degree. This chapter presents a systematic sensitivity analysis of FINUFFT's configuration space. We vary each parameter independently while holding others fixed, and measure the resulting effect on execution time across a representative range of inputs.
