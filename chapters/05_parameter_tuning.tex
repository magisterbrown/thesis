\section{Overview}

With $\sigma$ addressed in the previous chapter, this chapter turns to the remaining tunable parameters. The goal is to develop a unified heuristic that selects near-optimal values for all parameters simultaneously, with low planning overhead.

\section{Parameters Under Consideration}

Beyond $\sigma$, the parameters considered in this chapter are:

\begin{itemize}
    \item Spreading kernel width $w$ (when not fully determined by $\sigma$ and $\varepsilon$)
    \item FFT plan strategy (FFTW \texttt{ESTIMATE}, \texttt{MEASURE}, \texttt{PATIENT}, etc.)
    \item Number of threads
    \item Chunking and batching parameters for multi-transform execution
\end{itemize}
